# Iris Flower Classification 🌸

## Overview

This project focuses on classifying Iris flowers into three different species using Machine Learning techniques. The Iris dataset is one of the most popular datasets for beginners in Data Science and Machine Learning.

The goal is to build a classification model that can accurately predict the species of an Iris flower based on its physical characteristics.

## Dataset

The Iris dataset contains 150 samples of iris flowers belonging to three species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

### Features

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

### Target Variable

* Species of Iris Flower

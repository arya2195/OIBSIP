# Sales Price Prediction 📈

## Overview

This project focuses on predicting sales prices using Machine Learning techniques. By analyzing historical sales data and various influencing factors, the model can estimate future sales prices and provide valuable insights for business decision-making.

Sales price prediction is widely used in retail, e-commerce, and business analytics to optimize pricing strategies and improve revenue forecasting.

## Objective

The main objectives of this project are:

* Analyze sales data and identify factors affecting sales prices.
* Build a Machine Learning model for price prediction.
* Evaluate model performance using regression metrics.
* Gain practical experience in predictive analytics.

## Dataset

The dataset contains sales-related information such as:

* Product Features
* Sales Amount
* Product Category
* Market Information
* Historical Pricing Data
* Other Relevant Variables

### Target Variable

* Sales Price

## Project Workflow

### 1. Data Collection

* Loaded and explored the dataset.
* Examined dataset structure and feature information.

### 2. Data Preprocessing

* Handled missing values.
* Removed duplicate records.
* Processed categorical and numerical features.
* Prepared data for model training.

### 3. Exploratory Data Analysis (EDA)

* Analyzed feature distributions.
* Identified relationships between variables and sales prices.
* Visualized important trends using charts and graphs.

### 4. Feature Engineering

* Selected important features.
* Encoded categorical variables.
* Improved dataset quality for model training.

### 5. Model Building

* Split data into training and testing sets.
* Trained a Machine Learning regression model.
* Optimized model performance.

### 6. Model Evaluation

Evaluated the model using:

* R² Score
* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)

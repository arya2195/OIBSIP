# Car Price Prediction 🚗

## Overview

This project aims to predict the selling price of a car using Machine Learning techniques. The model analyzes various features of a vehicle such as brand, year, fuel type, transmission type, mileage, and engine specifications to estimate its market price.

Car price prediction is a practical application of regression algorithms and helps buyers and sellers make informed decisions based on vehicle characteristics.

## Objective

The main objectives of this project are:

* Analyze car-related data and identify factors affecting car prices.
* Build a Machine Learning model to predict car prices.
* Evaluate model performance using regression metrics.
* Gain hands-on experience with predictive analytics.

## Dataset

The dataset contains information about used cars, including:

* Car Name
* Year
* Selling Price
* Present Price
* Kilometers Driven
* Fuel Type
* Seller Type
* Transmission Type
* Owner Information

### Target Variable

* Selling Price

## Project Workflow

### 1. Data Collection

* Imported the car dataset.

### 2. Data Preprocessing

* Checked for missing values.

### 3. Exploratory Data Analysis (EDA)

* Analyzed relationships between features and car prices.
* Identified important factors influencing vehicle value.
* Visualized trends and distributions.

### 4. Model Building

* Split data into training and testing sets.
* Trained a Machine Learning regression model.
* Optimized model performance.

### 5. Model Evaluation

Evaluated the model using:

* R² Score
* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
